Searching

Search returns assays containing one or more of the search terms.

Example searches (on the 'name' field by default):

  • "parylene n"
  • parylene AND "para tech"
  • parylene OR PCTFE
  • parylen?
  • pary*

Specify a different field:

  • experiment:majorana
  • practitioner:smith
  • technique:naa
  • technique:(gamma OR naa) AND name:plastic

Supported search fields are:

  • grouping/experiment
  • name
  • description
  • id
  • source
  • owner
  • institution
  • technique
  • date
  • requestor
  • practitioner

Enter 'all' to return all assays.

Display

Use the plus and minus buttons to expand individual assays. Click 'Expand all' to exapnd them all
