Searching

Search returns assays containing one or more of the search terms.

You can alter this default behavior using wildcards and operators such as:

  • "stainless steel"
  • stainless AND steel
  • stainless OR steel
  • stee?
  • stainless s*

Enter 'all' to return all assays.

Display

Use the plus and minus buttons to expand individual assays. Click 'Expand' to exapnd them all

Click 'Detail' to toggle the global detail level.
